# Project Understanding: elkaero Personal Web Space

## Overview
A personal website for elkaero - a professional web presence to showcase articles, projects, and interesting material. Aimed at potential employers while being personal and engaging.

## Key Requirements

### Pages (5 total)
1. **Landing Page** - Introduction, "Hello this is my Space", what elkaero does, navigation to other sections
2. **Writing Page** - Articles/publications
3. **Projects Page** - Project showcases
4. **Visuals Page** - Visual content
5. **About Page** - 2-3 paragraphs about elkaero
6. **Contact Page** - Contact info/email
7. **Footer** - Must connect to GitHub

### Features
- Static site (no database, no CMS)
- Plain HTML, CSS, possibly JS

### Design
- **Colors**:
  - #8DBCC7
  - #A4CCD9
  - #C4E1E6
  - #EBFFD8
- **Fonts**: Rubik Light 300 (paragraphs), Rubik Regular 400 (headings)
- **Style**: Modern, minimalist, but not clean
- **Responsive**: Mobile first (standard breakpoints)

### Technical Requirements
- **Stack**: Plain HTML, CSS, JS (no frameworks)
- **Hosting**: Debian server
- **Performance**: Fast loading (static), 500 concurrent users, 99.5% availability
- **Security**: SSL/TLS, SSH authentication, Git backup

### Languages
- English only

### Non-Functional
- No SEO required
- No Google Analytics
- No external integrations

## Goals
1. Make elkaero enjoy writing and creating projects
2. Impress potential employers
3. Long-running, growing website
